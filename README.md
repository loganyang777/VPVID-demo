# VPVID Demo

Demo website for **VPVID: Variance-Preserving Velocity-Guided Interpolant Diffusion for Speech Enhancement and Dereverberation**.

## Overview

This repository contains an interactive demo showcasing audio samples and spectrograms comparing different speech enhancement methods:

- **Clean (Target)**: Original clean speech
- **Noisy (Input)**: Noisy input speech
- **Baseline Methods**: SGMSEP, VPIDM, FLOWSE
- **Our Methods**: VPVID, VPVID-PC, VPVID-ODE
